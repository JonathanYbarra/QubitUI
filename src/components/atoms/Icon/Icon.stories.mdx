# **Sizes**
For larger or smaller icon, you can use the `size` prop. the default size is `md`.

**`xs`** has 1.875rem heigth.

**`md`** has 2.5rem heigth.

**`lg`** has 3rem heigth.

**`xl`** has 3rem heigth.
